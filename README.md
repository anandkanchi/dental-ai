# Dental AI Radiograph Analysis

This project detects dental anomalies from OPG radiographs.

## Project Structure

frontend/
User interface for login, patient details, and report generation.

backend/
Python API handling anomaly detection, tooth numbering, and model inference.

## Model
Keras based multi-class classification model trained on dental radiograph datasets.

## Features
- Tooth numbering
- Missing tooth detection
- Anomaly detection
- Automated dental report generation
